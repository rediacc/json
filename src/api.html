API Documentation

RESTful JSON API for accessing template and configuration data

GET /templates.json

Returns the complete templates index with metadata for all available templates.

Response Format:

{
  "version": "1.0",
  "generated": "2024-01-01T00:00:00Z",
  "total_templates": 22,
  "categories": ["api", "auth", "cache", "db", ...],
  "templates": [
    {
      "id": "template_name",
      "name": "Display Name",
      "description": "Template description",
      "category": "category_name",
      "tags": ["tag1", "tag2"],
      "file_count": 4,
      "has_readme": true,
      "has_docker": true,
      "download_url": "templates/template_name.json"
    }
  ]
}

GET /templates/{template-id}.json

Returns detailed information and file contents for a specific template.

Example URLs:

  • /templates/api-gateway_kong.json
  • /templates/databases_mysql.json
  • /templates/caching_redis.json

Response Format:

{
  "id": "template_name",
  "name": "Display Name",
  "description": "Template description",
  "category": "category_name",
  "tags": ["tag1", "tag2"],
  "files": [
    {
      "name": "docker-compose.yaml",
      "path": "docker-compose.yaml",
      "type": "yaml",
      "content": "file contents here..."
    }
  ],
  "readme": "README.md contents...",
  "generated": "2024-01-01T00:00:00Z"
}

GET /configs/pricing.json

Returns pricing configuration data including tiers and professional services.

GET /configs/services.json

Returns services configuration data.

GET /configs/tiers.json

Returns tier configuration data.

Usage Examples

JavaScript/Fetch

// Load templates index
const templates = await fetch('/templates.json').then(r => r.json());

// Load specific template
const template = await fetch(`/templates/${templateId}.json`).then(r => r.json());

// Get file content
const dockerCompose = template.files.find(f => f.name === 'docker-compose.yaml')?.content;

curl

# Get templates index
curl https://json.rediacc.com/templates.json

# Get specific template
curl https://json.rediacc.com/templates/api-gateway_kong.json
